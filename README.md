# 1.주제  
## 테니스코트 커뮤니티
# 2.프로젝트 목적
### javafx를 활용하여  웹페이지와 흡사한 형태로 운동센터 커뮤니티를 제작  
프로그램 수요자 입장에서는 예약관리와 매출 통계를 확인할 수 있고   
대회를 개최하여 게시판 형태로 이를 등록하여   
회원들의 참가를 유도해 회원유치에 도움이 되도록 하였고,  
수요자의 고객들(회원)은 소통게시판을 통하여 회원간 소통이 가능할 수 있게끔 하였습니다.
